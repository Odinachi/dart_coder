## **Setup & Installation**

1. **Clone the Repository**
   ```sh
   git clone https://github.com/Odinachi/dartcoder.git
   cd dartcoder
   ```  

2. **Install Dependencies**
   ```sh
   flutter pub get
   ```  

3. **Run the App**
   ```sh
   flutter run
   ```
